# ADAS Event Mining and KPI Dashboard

## Goal
Build a beginner-friendly automotive analytics pipeline that processes vehicle telemetry, detects driving events, computes KPIs, and visualizes them in an interactive dashboard.

## Tech Stack
Python, Pandas, Scikit-learn, Streamlit, GitHub

## Planned Features
- Data ingestion from simulated and public automotive datasets
- Event detection for braking, acceleration, and steering
- KPI computation
- Risk scoring
- Interactive dashboard
